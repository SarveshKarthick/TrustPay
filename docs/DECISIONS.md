\# Architecture Decisions



\## AD-001

Decision: Mobile-first application



Reason:

Most freelancers and clients use smartphones for communication and payments.



\---



\## AD-002

Decision: FastAPI for backend



Reason:

High performance, automatic OpenAPI documentation, async support, and Python ecosystem.



\---



\## AD-003

Decision: PostgreSQL instead of MongoDB



Reason:

Escrow transactions require strong consistency, ACID compliance, and relational integrity.
